# TasksExpress

### Basic task manager started with:

+ Node.js
+ Express
+ MongoDb
+ Socket.io

<p>Register a list of tasks in real time with the server.</p>
<p>Installation:</p>

```
git clone https://github.com/ThiagoMartinsdeMelo/TasksExpress.git
```
